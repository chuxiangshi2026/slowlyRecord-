# 慢记单词本 - 使用说明书

## 软件简介

**慢记单词本**是一款基于**艾宾浩斯遗忘曲线**开发的智能单词记忆辅助工具，以 uTools 插件形式运行。它通过科学的记忆算法，帮助用户在最佳时间点复习单词，达到长期记忆的效果。

### 核心特点

- **科学记忆算法**：基于艾宾浩斯遗忘曲线，自动安排复习计划
- **多种添加方式**：支持输入、划词、划段、截图等多种方式添加单词
- **多引擎翻译**：支持有道、百度、阿里、腾讯、DeepSeek、Kimi、通义千问、智谱 GLM、uTools AI、OpenAI、Gemini、Claude、Ollama 本地模型等十余种翻译引擎
- **离线使用**：支持离线词典，无网络也能正常使用
- **专注模式**：迷你悬浮窗口，支持置顶、贴边、透明度调节
- **丰富词库**：内置 CET-4/6、考研、托福、雅思、GRE、GMAT 等权威词库
- **多元记忆**：支持单词、数字、文本、快捷键等多种记忆训练
- **数据安全**：本地数据库存储，支持导入导出

## 安装与启动

### 环境要求

- **uTools** 软件（Windows / macOS / Linux）
- 网络连接（首次配置翻译引擎时需要）

### 安装步骤

1. 下载并安装 [uTools](https://u.tools/)
2. 在 uTools 插件市场搜索"慢记单词本"
3. 点击安装插件
4. 安装完成后，按 `Alt + 空格` 呼出 uTools，输入 `慢记` 或 `mj` 即可使用

## 添加单词

### 1. 输入添加

1. 呼出 uTools，输入任意单词（如 `apple`）
2. 选择"**加入单词簿**"
3. 系统自动翻译并弹出翻译结果
4. 点击确认即可加入学习列表

### 2. 划词添加

1. 选中任意文本（单词或短语）
2. 呼出 uTools，输入 `划词添加` 或 `hc`
3. 选择翻译引擎，确认后自动添加

### 3. 划段添加

1. 选中一段文字
2. 呼出 uTools，输入 `划段添加`
3. 系统会将段落自动拆分为单词并加入学习列表

### 4. 截图添加

1. 呼出 uTools，输入 `截图添加` 或 `ss`
2. 截取包含文字的图片区域
3. 系统自动 OCR 识别图片中的文字
4. 选择需要添加的单词，确认后添加

> **注意**：截图翻译每日限制 5-10 次（未配置个人密钥时）。配置自己的密钥后可无限使用。

## 单词复习

### 进入复习模式

呼出 uTools，输入 `mj`、`慢记` 或 `review`，进入单词列表界面。

### 复习界面说明

| 区域 | 说明 |
|------|------|
| **单词卡片** | 显示单词、音标（词根词缀颜色区分）、发音按钮 |
| **释义区** | 显示中文释义，可隐藏/显示 |
| **操作按钮** | 认识 / 模糊 / 忘记、置顶、置底、永久记住 |
| **单词筛选** | 支持按复习状态、等级、词库筛选 |
| **词库切换** | 支持多词库切换学习 |

### 复习操作

| 操作 | 说明 | 快捷键 |
|------|------|--------|
| **播放发音** | 点击喇叭图标播放美式发音 | `Ctrl + P` |
| **显示/隐藏释义** | 点击眼睛图标切换 | `Ctrl + H` |
| **认识** | 标记为已掌握，升级复习等级 | `→` / `D` |
| **模糊** | 标记为模糊，保持当前等级 | `↓` / `S` |
| **忘记** | 标记为忘记，降低复习等级 | `←` / `A` |
| **永久记住** | 该单词不再出现在复习列表 | - |
| **编辑释义** | 点击编辑按钮修改释义 | - |

### 记忆等级说明

单词共有 **12 个等级**，每次有效复习会提升等级，延长下次复习时间：

| 等级 | 复习间隔 | 说明 |
|------|----------|------|
| 0 | 立即 | 新添加的单词 |
| 1 | 5 分钟 | 第一次复习 |
| 2 | 30 分钟 | 第二次复习 |
| 3 | 6 小时 | 第三次复习 |
| 4 | 12 小时 | 第四次复习 |
| 5 | 1 天 | 第五次复习 |
| 6 | 2 天 | 第六次复习 |
| 7 | 4 天 | 第七次复习 |
| 8 | 1 周 | 第八次复习 |
| 9 | 半个月 | 第九次复习 |
| 10 | 1 个月 | 第十次复习 |
| 11 | 3 个月 | 第十一次复习 |
| 12 | 永久 | 完成长期记忆 |

> 达到 12 级的单词会自动标记为"**永久记住**"，可在"已记完"列表中查看。

## 听写模式

### 功能说明

听写模式通过播放单词发音，让用户拼写单词，检验听力与拼写能力。

### 使用方法

1. 在单词列表界面，点击"**听写模式**"按钮
2. 系统开始播放单词发音
3. 用户根据发音输入单词
4. 输入完成后回车确认，查看结果

### 听写设置

- **范围选择**：可选择全部单词、待复习单词、指定等级单词
- **播放次数**：设置每个单词发音播放次数
- **自动下一题**：答对后自动播放下一题
- **保存进度**：自动保存听写进度，下次可从上次位置继续

## 数字记忆训练

### 功能说明

数字记忆训练是一种记忆力锻炼游戏，通过记忆随机数字序列，提升短期记忆能力。

### 使用方法

1. 呼出 uTools，输入 `数字记忆`
2. 选择难度级别（4位、6位、8位...）
3. 点击"**开始训练**"
4. 记忆屏幕上显示的数字（限时显示）
5. 数字消失后，按顺序输入记忆中的数字
6. 查看记忆结果与准确率

### 训练模式

- **经典模式**：按顺序记忆数字
- **倒序模式**：倒序回忆数字
- **乱序模式**：按位置回忆指定数字

## 导入与导出

### 导出单词

1. 进入单词列表界面
2. 点击"**导出**"按钮
3. 选择导出范围：
   - 全部单词
   - 待复习单词
   - 未记住单词
   - 已记住单词
4. 选择导出格式（JSON / TXT / CSV）
5. 保存文件到本地

### 导入单词

1. 进入单词列表界面
2. 点击"**导入**"按钮
3. 选择导入文件（支持 JSON、TXT、CSV）
4. 系统自动解析并导入

### 文件格式说明

#### 格式一：简单格式（仅单词）

每行一个单词，系统会自动查询释义：

```
apple
banana
orange
computer
```

#### 格式二：完整格式（保留状态）

**JSON 格式**：

```json
[
  {
    "text": "apple",
    "explains": "n. 苹果",
    "phonetic": "[ˈæpl]",
    "level": 3,
    "isReview": true,
    "remember": false
  }
]
```

**TXT/CSV 格式**：

```
单词|释义|音标|等级|是否复习|是否记住
apple|n. 苹果|[ˈæpl]|3|true|false
```

## 常见问题

### Q1: 为什么截图翻译有次数限制？

截图翻译涉及 OCR 识别和翻译 API 调用，成本较高。未配置密钥时，每日限制 5-10 次。建议配置自己的 API 密钥，可享受免费额度。

### Q2: 如何离线使用？

在设置中选择"**离线词典**"作为翻译引擎，无需网络即可查词。但离线词典词汇量有限，建议配合在线引擎使用。

### Q3: 数据存储在哪里？

所有数据存储在本地数据库中，不会上传到云端。重装系统前请记得导出备份。

### Q4: 如何同步多台设备的数据？

目前暂不支持自动云端同步。可通过导出/导入功能手动同步：
1. 在设备 A 导出单词文件
2. 将文件传输到设备 B
3. 在设备 B 导入文件

### Q5: 为什么有些单词没有发音？

请检查：
1. 网络连接是否正常
2. 发音引擎设置是否正确
3. 是否启用了离线发音

### Q6: 如何更换主题？

插件自动适配 uTools 的主题设置。在 uTools 设置中切换浅色/深色主题，插件会自动跟随切换，支持暗黑模式。

### Q7: 添加单词后没有出现在列表中？

请检查：
1. 是否正确点击了"确认添加"
2. 列表筛选条件是否设置为"全部"
3. 是否开启了"只显示待复习"筛选

### Q8: 如何删除单词？

在单词列表中，右键点击单词卡片，选择"删除"。

### Q9: 如何切换词库？

在单词列表界面，点击顶部的"**切换词库**"下拉菜单，选择需要学习的词库。词库数据独立存储，切换后显示对应词库的单词。

### Q10: 听写模式如何保存进度？

听写模式会自动保存当前进度，下次进入时会询问是否继续上次的听写。也可在听写设置中关闭自动保存。
